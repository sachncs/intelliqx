# IntelliqX v1.0 — Release Notes

## Highlights

- **Full 4-tier agent architecture** — Planner, Orchestrator, Memory, Knowledge/RAG, Tool Manager (Tier 1); Requirements Intel, Code Intel, Risk, Test Design, Test Data, Coverage, Critic (Tier 2); Environment, Execution, Self-Healing, Failure Analysis, Design Intel (Tier 3); Observability, Reporting, Governance & Compliance, Release Readiness (Tier 4).
- **Multi-cloud portability** — same agent code runs on AWS, GCP, and Modal via adapter libs (no cloud SDK calls in agent code).
- **Knowledge Graph on Parquet + DuckDB** — file-based, no managed graph DB needed.
- **Vector search via zvec** — embedded, persisted to object storage, runs anywhere.
- **Self-healing selectors** — LLM-ranked candidate selectors with confidence thresholds.
- **Release readiness decisioning** — Go / Conditional Go / No-Go with explainability and confidence.
- **Audit + human-approval workflows** — RBAC + ABAC + immutable audit trail.
- **324 tests passing**, lint clean, contract tests enforce portability guarantees.

- **Local**: `uv sync && uv run pytest`
- **AWS**: `cd infra/aws/cdk && cdk deploy --all`
- **GCP**: `cd infra/gcp && cdktf deploy`
- **Modal**: `cd infra/modal && modal deploy`
